# Sherlock of Bugs 

AI-powered bug detection and debugging assistant that analyzes code, identifies potential issues, explains root causes, and recommends fixes.

## Live Demo

[https://huggingface.co/spaces/eva1267890/ChatBot2026](https://huggingface.co/spaces/c3team/sherlock-ob)

##  Features

- AI-powered code analysis
- Bug classification
- Error explanation
- Suggested fixes
- Debugging recommendations
- Code quality improvements
- Developer-friendly reports

## How It Works

1. User submits code/error logs
2. AI analyzes patterns and possible failures
3. System identifies likely bugs
4. Generates explanations and fixes

## Tech Stack

- Python
- Hugging Face Transformers
- LLM APIs
- Gradio
- Machine Learning
- Natural Language Processing


## Installation

```bash
git clone repo-url
pip install -r requirements.txt
python app.py
```
